# Roadmap

## Milestone 0: Repository Bootstrap

- Create repository and open-source project metadata.
- Define product design and data layout.
- Add an example novel folder skeleton.

## Milestone 1: Vault Project Skeleton

- Create a novel project under an Obsidian vault.
- Generate the standard folder layout.
- Read and write Markdown with YAML frontmatter.
- Build a local index cache.

## Milestone 2: Canon and Character MVP

- Canon editor.
- Character editor.
- Character arc matrix.
- OOC guardrails.

## Milestone 3: Timeline and Location MVP

- Forward-only timeline chain.
- Location graph.
- Basic travel feasibility check.

## Milestone 4: Outline to Section Generation

- Book, volume, arc, chapter, and section outlines.
- Prompt assembly from project state.
- Generate one section.
- Save run records.

## Milestone 5: Consistency Checks

- Canon check.
- Timeline check.
- Location check.
- Character check.
- Style guardrail check.

## Milestone 6: SillyTavern Interop

- Import character cards.
- Export character cards.
- Export canon as lorebook/world info.
- Import model preset concepts.

## Milestone 7: Versioning

- Git integration.
- Run comparison.
- Accepted draft history.
- Rollback and restore.
